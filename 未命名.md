# 🔥 二、最可能的 3 个原因（按概率排序）

---

## 🥇 原因1：你访问的是 localhost / 127.0.0.1 被浏览器绕过代理（最高概率）

Chrome 有一个“隐藏规则”：

```text
localhost / 127.0.0.1 默认不走系统代理
```

👉 结果就是：

- example.com → 走 Burp ✔
    
- DVWA（本机）→ 直接直连 ❌
    

---

## ✔ 解决（必须做）

### 👉 强制改访问方式：

不要用：

```text
http://127.0.0.1/DVWA
```

改用：

```text
http://127.0.0.1.local
```

或者最稳：

```text
http://lvh.me/DVWA
```

（lvh.me 会解析到 127.0.0.1，但不会被浏览器绕过）

---
